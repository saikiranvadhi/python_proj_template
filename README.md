# Python Project Template

This repository is a template for starting new Python projects. It includes a recommended structure, logging setup, environment variable management, and example modules.

## Getting Started

1. Clone this repository.
2. Rename the `src/project_name` folder to your desired project name.
3. Update `pyproject.toml` and other files to use your new project name.
4. Copy `.env.example` to `.env` and set your environment variables as needed.

## Features
- Python 3.11+
- Logging setup
- Environment variable management
- Example module structure

## Usage

Replace all instances of `project_name` with your actual project name throughout the codebase.

## Environment Variables

Add your environment variables to the `.env` file. See `.env.example` for a template.
